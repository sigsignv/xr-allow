# xr-allow

A CLI utility for allowing SSH connection IPs using the XREA API.

## Install

```sh
$ go install github.com/sigsignv/xr-allow
```

## Example

```toml
[[Server]]
account = "USERNAME"
server_name = "SERVERNAME.xrea.com"
api_secret_key = "YOUR_API_KEY"
```
